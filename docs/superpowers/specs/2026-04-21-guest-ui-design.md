## 1. 设计风格

**温暖家庭风 · 活力暖橙**

简约清爽风格，以温暖橙色为主色调，搭配米白背景，营造温馨亲切的家庭用餐氛围。

### 2.1 主色调

| 用途 | 色值 | 说明 |

### 2.2 页面色彩

## 7. 页面设计

### 7.1 菜单页 (pages/guest/menu)

**布局结构**：
- 顶部：分类标签横向滚动（高度 80rpx）
- 中部：菜品网格（2列），可下拉刷新
- 底部：购物车悬浮按钮

**分类标签**：
- 选中态：背景 `#FF8C42`，文字白色
- 未选中：背景 `#FFF5EE`，文字 `#666666`

**菜品卡片**：
- 尺寸：宽度 (750rpx - 48rpx - 16rpx) / 2 = 343rpx
- 图片高度：180rpx，圆角 `12rpx`
- 卡片圆角：`16rpx`
- 卡片内边距：`20rpx`
- 价格颜色：`#FF8C42`

**购物车悬浮按钮**：
- 位置：右下角，距底部 `160rpx`，距右边 `24rpx`
- 尺寸：`100rpx × 100rpx`
- 背景：白色 + 橙色阴影
- 图标：购物车 emoji (🛒)
- 徽章：右上角红色圆点

### 7.2 购物车页 (pages/guest/cart)

**布局结构**：
- 顶部：清空按钮 + 商品数量
- 中部：商品列表
- 底部：结算栏（固定定位）

**商品项**：
- 图片：60rpx × 60rpx，圆角 `8rpx`
- 数量调整：- / 数量 / + 按钮
- 删除按钮：红色 `#FF4D4F`

**结算栏**：
- 高度：`100rpx`
- 背景：白色
- 合计金额：橙色 `#FF8C42`，字号 `32rpx`，bold
- 结算按钮：背景 `#FF8C42`，文字白色，圆角 `20rpx`

### 7.3 确认订单页 (pages/guest/confirm)

**布局结构**：
- 订单商品列表
- 备注输入框
- 底部提交栏

**备注输入框**：
- 背景：`#FFF9F5`
- 圆角：`12rpx`
- placeholder：`添加备注（如：少辣、不要香菜）`

**提交按钮**：
- 全宽，高度 `80rpx`
- 背景：`#FF8C42`
- 圆角：`20rpx`
- 文字：白色，32rpx，bold

### 7.4 订单列表页 (pages/guest/orders)

**订单卡片**：
- 背景：白色
- 圆角：`16rpx`
- 内边距：`24rpx`
- 阴影：`0 2rpx 8rpx rgba(0, 0, 0, 0.06)`

**状态标签**：
- 待支付：背景 `#FFF5EE`，文字 `#FAAD14`
- 已支付：背景 `#E6F4FF`，文字 `#1677FF`
- 制作中：背景 `#F4F0FF`，文字 `#722ED1`
- 已完成：背景 `#F6FFED`，文字 `#52C41A`
- 已取消：背景 `#F5F5F5`，文字 `#999999`

**操作按钮**：
- 取消订单：背景 `#F5F5F5`，文字 `#666666`
- 去支付：背景 `#FF8C42`，文字白色

### 7.5 订单详情页 (pages/guest/order-detail)

**状态展示**：
- 状态图标：大圆圈 100rpx
- 状态文字：橙色 `#FF8C42`
- 预估时间：灰色 `#999999`

**信息卡片**：
- 背景：白色
- 圆角：`16rpx`
- 内边距：`24rpx`

### 7.6 个人中心页 (pages/guest/profile)

**用户信息区**：

**统计卡片**：
