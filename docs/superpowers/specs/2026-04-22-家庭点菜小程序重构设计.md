## 三、Zustand Store 设计

### 3.1 authStore

```typescript
interface UserInfo {
  id: string
  username?: string
  nickname?: string
  avatar?: string
  role: 'admin' | 'merchant' | 'user'
  status?: string
  openId?: string
}

interface AuthState {
  isLogin: boolean
  token: string | null
  userInfo: UserInfo | null
  role: 'guest' | 'chef' | null
}

interface AuthActions {
  login: (token: string, userInfo: UserInfo, role: 'guest' | 'chef') => void
  logout: () => void
  updateUserInfo: (userInfo: UserInfo) => void
}

type AuthStore = AuthState & AuthActions
```

**功能:**
- 登录/登出
- 用户信息更新
- 自动从 Storage 恢复状态
- 持久化到 Storage

### 3.2 cartStore

```typescript
interface CartItem {
  dishId: number | string
  dishName: string
  price: number
  quantity: number
  image?: string
}

interface CartState {
  items: CartItem[]
}

interface CartActions {
  addToCart: (item: CartItem) => void
  removeFromCart: (dishId: number | string) => void
  updateQuantity: (dishId: number | string, quantity: number) => void
  clearCart: () => void
  setCart: (items: CartItem[]) => void
}

interface CartComputed {
  totalCount: number
  totalAmount: number
}

type CartStore = CartState & CartActions & CartComputed
```

**功能:**
- 添加/移除/更新购物车商品
- 计算总数量和总金额
- 自动持久化到 Storage

### 3.3 Store 实现要点

- 使用 `persist` middleware 实现 Storage 持久化
- 每个 store 独立文件，职责单一
- 通过 `useAuth()`, `useCart()` hooks 访问
