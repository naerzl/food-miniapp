## 1. 产品概述

### 1.1 阶段目标

第三阶段目标：**前端集成、能力扩展**，完成 Web 管理后台和微信小程序的前端开发，接入真实微信支付，增加通知系统和细粒度权限控制。

### 1.2 核心目标

| 目标编号 | 目标描述 | 关键指标 | 优先级 |
|----------|----------|----------|--------|
| G1 | 完成前端集成 | Web后台和小程序完整可用 | P1 |
| G2 | 接入真实支付 | 完成微信支付接入 | P2 |
| G3 | 通知系统 | 订阅消息/邮件通知 | P2 |
| G4 | 权限细化 | 多角色细粒度权限控制 | P2 |
| G5 | 数据导出 | 支持Excel/CSV导出 | P3 |

## 4. API 接口清单

### 4.1 新增/增强接口

| 接口 | 方法 | 路径 | 说明 | 优先级 |
|------|------|------|------|--------|
